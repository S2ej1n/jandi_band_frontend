## ⚒️ 기술 스택
- Vite React
- TypeScript
- Zustand
- TanStack Query
- React Hook Form + Zod
- ESLint

## 📌 주요 기능

- 동아리 관리 : 동아리 생성, 가입, 멤버 관리
- 일정 관리 : 일정 등록 캘린더 제공
- 팀 관리 : 팀 생성 및 연습 스케줄 관리
- 곡 투표 시스템 : 곡 목록 작성 및 투표
- 공연 홍보 게시판 : 동아리 홍보 글 작성 삭제 및 댓글, 카카오 맵을 이용한 장소 조회
- 초대 시스템 : 동아리/팀 초대 링크 생성 및 카카오톡 메세지 공유
- 인증/인가: JWT 기반 인증, 카카오 소셜 로그인
- 반응형 UI : 모바일/데스크톱 지원
